# Identidad Astral - Landing Page Premium

## Luxury Editorial - Cinematic Aesthetic

Proyecto desarrollado con:
- **React 19**
- **Tailwind CSS**
- **GSAP 3** (Animaciones cinemáticas)
- **Vite** (Build Tool)

### Experiencia Visual
- **Tipografía**: Cormorant Garamond & Plus Jakarta Sans.
- **Textura**: Overlay de ruido procedimental para sensación de papel.
- **Interacciones**: Cursor magnético, telemetría en vivo y animaciones de scroll high-fidelity.

### Instalación
```bash
npm install
npm run dev
```
